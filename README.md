## SESC Finance Microservice

This laravel App serves as one of the 3 Microservices built for a university modules. the Goal was a set of microservices that would form 3 parts of a University Library Service, this would handle the Finance data, and would communicate with the [library](https://github.com/LEDMann/laravel-library) and [student](https://github.com/LEDMann/laravel-student) Microservices for the information and functions that are not within the scope of this microservice.
